Roadmap

This section serves as a way for me to inform you about the current development state of some new features and bugs fixes.

A feature can be in one of the following three "levels" of development:

  • In mind: Features that would be great to integrate into the asset.
  • In Development: The feature is in development (this doesn't mean that the feature will be ready).
  • Ready: It's done! this is going to the next release.

Features

In mind

  • Core
  • Implementation
    • Character actions asset.
    • Footsteps with audio.

In Development

  • Core
    • One Way platforms for 2D and 3D.
  • Implementation
    • First person camera mode.

Ready

  • Core
    • Kinematic movement (without collision detection).
  • Implementation
    • Ladder climbing.
    • Rope climbing (vertical only).
    • Auto follow camera (circular).
    • Root motion.
    • Better animation system (integrated with the state controller component).
    • Better properties for Surfaces and Volumes (add velocity, set velocity, set body size, etc). This is useful for jump pads.
